# Discord Bot

A Discord bot that you can run yourself.

## What You Need

- Python 3.8 or higher
- A Discord account
- A bot token from Discord Developer Portal

## Setup

### 1. Get Your Bot Token

1. Go to https://discord.com/developers/applications
2. Click "New Application" and name it
3. Go to the "Bot" tab on the left
4. Click "Add Bot"
5. Click "Reset Token" and copy the token

### 2. Invite Bot to Your Server

1. In the Developer Portal, go to "Installer" > and then copy the link it gave you into your browser
2. Select "bot" and "Administrator" under Scopes and Permissions
3. Copy the generated URL and paste it in your browser
4. Select your server and click "Authorize"
